Colorado Airshow Database by fdorigo

A simple web application for the Colorado Aviation Group, used to register participant to air shows. The application is implemented in JSP and uses a MySQL database as a back end.

The application uses the FAA database to speed up data entry for US registered planes that participate in the airshow. It is also designed to help tracking the scores that each judge assigns to the planes that have been designated to be judged.
